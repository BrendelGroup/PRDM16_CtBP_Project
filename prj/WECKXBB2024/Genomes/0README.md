# Get the genomes!

The download and indexing of the mouse and Drosophila genomes is a prerequisite step to the downstream analyses and can be accomplisheds as follows (read the 0Record files for details):

```bash
cd genomeMM39
bash ./0Record 
cd ..

cd genomeDM6/
bash ./0Record 
cd ..

cd ../genomeMM39DM6/
bash ./0Record 
cd ..
```
